## Expert Analysis Framework

### Phase 1: Transcript Processing & Structure Mapping

**Step 1: Load and Analyze Transcript**
- Read the complete markdown transcript file at the specified path
- Identify structural markers (timestamps, section breaks, transitions)
- Map the complete flow from beginning to end

**Step 2: Structural Segmentation**
- Break the transcript into distinct structural sections
- Identify opening hooks, core content sections, and closing patterns
- Map transition techniques between sections

### Phase 2: Structural DNA Extraction

**A. Complete Structural Mapping (Analyze the ENTIRE script)**
Map the script into these exact structural components:
- **Opening Pattern**: First 15-30 seconds structure
- **Transition 1**: How it moves from hook to main content
- **Core Content Structure**: Main body organization pattern
- **Sub-section Transitions**: How each point connects to the next
- **Climax Building**: How tension/value builds throughout
- **Peak Moment**: Where the maximum value/impact lands
- **Wind-down Pattern**: How it resolves toward the end
- **CTA Integration**: How call-to-action is structurally integrated

**B. Universal Pattern Identification**
For EACH section, identify:
- **Sentence Structure Patterns**: Length, complexity, rhythm
- **Question Placement**: Where questions appear and their function
- **Information Flow**: How information is sequenced
- **Tension-Release Cycles**: Pattern of building and resolving
- **Pacing Markers**: Where speed changes occur
- **Repeat Formulas**: Any recurring structural elements
- **Transition Techniques**: How sections connect seamlessly
