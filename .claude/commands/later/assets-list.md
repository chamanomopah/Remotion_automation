# Assets List - Gera lista de assets necessários para o vídeo
# Uso: /assets-list [script-file]
# Exemplo: /assets-list "spider-man-script.md"

Analisa o roteiro e gera lista completa de assets visuais necessários:

## Input:
Arquivo de Roteiro: {script-file}

## Tipos de Assets Identificar:

### Imagens de Quadrinhos
- Capas específicas (ex: Amazing Fantasy #15)
- Páginas importantes
- Painéis específicos mencionados
- Cenas de batalha

### Imagens de Filmes/Séries
- Posters de filmes
- Stills específicos
- Cenas marcantes

### Elementos Gráficos
- Logotipos (Marvel, DC)
- Ícones de personagens
- Símbolos e emblemas

### Backgrounds
- Temáticos por personagem
- Cenários importantes
- Texturas e padrões

## Formato de Saída:
- Categoria do asset
- Descrição específica
- Fonte/sugestão de onde encontrar
- Prioridade (essencial/importante/opcional)
- Dimensões recomendadas

## Output:
Lista organizada por ordem de aparição no roteiro

Analise o roteiro e gere a lista completa agora.
