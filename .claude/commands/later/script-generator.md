# Script Generator - Transforma research em roteiro estilo Alt Shift X
# Uso: /script-generator [tema] [personagem/conceito]
# Exemplo: /script-generator "Spider-Man" "Peter Parker"

Gere um roteiro completo no estilo Alt Shift X seguindo a estrutura universal de 7 partes:

## Estrutura Exigida:
1. **Opening Hook (0:00-0:30)**: Authority → Comparative Frame → Curiosity Gap → Value Promise
2. **Foundation Setting (0:30-2:30)**: Context → Comparative Divergence → Core Question → Metaphorical Hook
3. **Background Deep Dive (2:30-8:00)**: Origin Story → Environmental Context → Progressive Revelation → Pattern Establishment
4. **Development Arc (8:00-15:00)**: Multi-Phase Analysis → Turning Points → Contradiction Exploration → Evidence Integration
5. **Comparative Reinforcement (15:00-16:30)**: Version Contrast → Implication Analysis → Broader Theme Connection
6. **Future Speculation (16:30-17:00)**: Predictive Analysis → Possibility Exploration → Final Question
7. **Resolution & CTA (17:00-17:30)**: Synthesis → Authority Reinforcement → Value Reminder → Natural CTA

## Input do usuário:
Tema: {tema}
Foco: {personagem/conceito}

## Requisitos:
- Duração: 17-20 minutos
- Tom: Autoritário mas acessível (estilo Alt Shift X)
- Incluir citações específicas e evidências
- Usar metáforas poderosas
- Criar tensão entre "o que as pessoas pensam" vs "a realidade"
- Conectar com experiências humanas universais
- Incluir pontos específicos para assets visuais (imagens de quadrinhos, cenas, etc.)

## Formato de saída:
Roteiro completo com timestamps e indicadores visuais: [TIMESTAMP] CONTEÚTO

Por favor, gere o roteiro completo agora.
