# Storyboard Minimum - Estrutura mínima de storyboard para cada vídeo
# Uso: /storyboard-minimum [script-file]
# Exemplo: /storyboard-minimum "spider-man-script.md"

Gera storyboard mínimo baseado na estrutura Tyrion Lannister para garantir todos elementos essenciais:

## Input:
Arquivo de Roteiro: {script-file}

## Estrutura Mínima Exigida (7 partes):

### 1. Opening Hook (0:00-0:30)
- [ ] Imagem de autoridade/credibilidade
- [ ] Elemento comparativo inicial
- [ ] Visual da "curiosity gap"
- [ ] Transição para promessa de valor

### 2. Foundation Setting (0:30-2:30)
- [ ] Contexto visual do tema
- [ ] Elemento que mostra divergência
- [ ] Representação da pergunta central
- [ ] Metáfora visual principal

### 3. Background Deep Dive (2:30-8:00)
- [ ] Origin story visual
- [ ] Contexto ambiental
- [ ] Elementos formativos
- [ ] Padrão estabelecido visualmente

### 4. Development Arc (8:00-15:00)
- [ ] Fases visuais distintas
- [ ] Turning points marcados
- [ ] Contradições exploradas
- [ ] Evidências integradas

### 5. Comparative Reinforcement (15:00-16:30)
- [ ] Contraste visual claro
- [ ] Implicações mostradas
- [ ] Conexão temática visual

### 6. Future Speculation (16:30-17:00)
- [ ] Estado atual visual
- [ ] Projeções futuras
- [ ] Cenários possíveis

### 7. Resolution & CTA (17:00-17:30)
- [ ] Síntese visual
- [ ] Elemento de autoridade final
- [ ] Lembrete de valor
- [ ] CTA natural integrado

## Formato de Saída:
- Checklist visual por seção
- Descrição do que mostrar em cada momento
- Coordenação com áudio/roteiro
- Tempo aproximado por elemento

## Requisitos:
- Mínimo 1 elemento visual por 30 segundos
- Transições suaves entre seções
- Consistência visual maintained
- Apoio direto ao conteúdo narrativo

Gere o storyboard mínimo completo agora.
