## Tipos de Cena Disponíveis (Personalizados):

### character-evolution
- Comparação lado a lado: versão antiga vs moderna
- Animação de transformação progressiva
- Labels com períodos/tempo
- Zoom em detalhes específicos de cada era

### theme-comparison
- Split screen vertical/horizontal
- Elementos comparativos animados
- Indicadores visuais de diferenças
- Transição suave entre focus points

### timeline-evolution
- Linha do tempo horizontal interativa
- Marcadores com eventos importantes
- Zoom progressivo ao longo da timeline
- Imagens/icons em cada marcador

### evidence-showcase
- Centralização em evidência específica
- Zoom detalhado em elemento crucial
- Texto explicativo animado
- Contexto visual de support

### relationship-map
- Nós representando elementos/personagens
- Conexões animadas mostrando relações
- Cores por tipo de relação
- Zoom e focus em conexões específicas

### cultural-impact
- Montagem rápida de influências
- Comparação com outros elementos culturais
- Gráficos/estatísticas animados
- Visualização de alcance/legado

## Requisitos Técnicos:
- Usar sistema VirtualCamera do projeto
- Manter consistência visual com resto do vídeo
- Otimizado para performance (máximo 1080p)
- Coordenar com assets existentes em /public
- Timing preciso baseado em {duration}
- Transições suaves entre cenas

## Integração com Projeto:
- Respeitar sistema de cores definido
- Usar fontes customizadas do projeto
- Manter aspecto ratio consistente
- Coordenar com narração/áudio
- Seguir ritmo visual estabelecido

## Output Esperado:
- Código .tsx completo da cena
- Instruções de integração no projeto
- Lista de assets necessários específicos
- Timing de animações detalhado
